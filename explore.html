Cas d'étude complet : Analyse des téléchargements PyPI (GCP & Streamlit)

Découvrez ma méthodologie complète pour la réalisation d'un pipeline de données scalable et interactif.

🎯 Contexte & Objectifs

Ce projet présente une analyse des données publiques issues de PyPI sur le nombre de téléchargements de packages Python, avec un pipeline automatisé réalisé sur GCP et Streamlit.

  • ✔️ Collecter et ingérer des données via Composer (Airflow)
  • ✔️ Transformation et agrégation via BigQuery SQL
  • ✔️ Dashboard interactif avec Streamlit
  • ✔️ CI/CD automatisé avec GitLab et Docker

⚙️ Workflow & DevOps (CI/CD)

J'ai utilisé GitLab CI/CD pour automatiser le déploiement via Docker et GCP Cloud Run.

Gestion des branches distinctes (prod & dev) permettant une séparation claire entre l'environnement de test et de production.
